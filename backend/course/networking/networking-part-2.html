üèì Ping - Test Connection Speed

Like playing catch with a server to see how fast it responds

# Windows/Mac/Linux Command Prompt
ping google.com

# Example Output:
PING google.com (172.217.14.110): 56 bytes
64 bytes from 172.217.14.110: time=12ms
64 bytes from 172.217.14.110: time=11ms
64 bytes from 172.217.14.110: time=13ms

# What this means:
# 12ms = Time for round trip to Google
# Lower = Better (under 50ms is good)

üó∫Ô∏è Traceroute - Map Your Data's Journey

Shows every router your data passes through on its way to destination

# Windows: tracert google.com
# Mac/Linux: traceroute google.com

# Example Output:
1 192.168.1.1 1ms (Your router)
2 10.0.0.1 15ms (ISP local)
3 72.14.233.76 25ms (ISP backbone)
4 108.170.248.97 45ms (Google edge)
5 172.217.14.110 48ms (Google server)

# Each line = one "hop" in your data's journey!

üîç Nslookup - DNS Detective

Investigate how domain names get translated to IP addresses

# Windows/Mac/Linux
nslookup google.com

# Example Output:
Server: 8.8.8.8
Address: 8.8.8.8#53

Non-authoritative answer:
Name: google.com
Address: 172.217.14.110

# Shows which DNS server answered
# and what IP address was returned

üìä Netstat - See Active Connections

Shows all network connections your computer is currently making

# Windows/Mac/Linux
netstat -an

# Example Output:
Active Connections
Proto Local Address Foreign Address State
TCP 192.168.1.100:80 172.217.14.110:443 ESTABLISHED
TCP 192.168.1.100:81 151.101.1.140:443 ESTABLISHED

# Shows what your computer is connected to
# ESTABLISHED = active connection
